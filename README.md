DOM Manipulation Example
This project demonstrates how to manipulate the Document Object Model (DOM) using JavaScript. It showcases the following features:

Dynamically changing text content.

Modifying CSS styles via JavaScript.

Adding and removing elements when a button is clicked.

HTML (index.html)
The HTML file contains:

A header with the title and a description.

A section with some initial content (title and text) that will be dynamically updated via JavaScript.

A button that triggers the DOM manipulation when clicked.

JavaScript (script.js)
The JavaScript file performs the following actions when the button is clicked:

Changes the text content of the title and paragraph.

Modifies the CSS styles of the title and paragraph.

Dynamically creates a new element and appends it to the page.

Hides the button after it's clicked to prevent further interaction.

How to Use
Clone the repository or download the files.

Open index.html in your browser.

Click the "Click Me!" button to see the changes in action.

Requirements
A modern web browser (e.g., Chrome, Firefox, Safari, etc.).
